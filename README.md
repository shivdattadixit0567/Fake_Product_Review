# Fake Review Product detection

# Project Description : 

* Projects aims to classify the Review

* Python

* Streamlit

# Install and Run

* We craete a venv environment to run the program
* conda create -p venv python==3.7 -y
* conda activate venv/
* python app.py
* streamlit run app.py
